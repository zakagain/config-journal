# README

## 📓 What is Config Journal?

_Config Journal_ is a plain-text (markdown) archive of my personal digital setup—focused on simplicity, privacy, and efficiency. No configs to clone, no scripts to run—just clean markdown notes documenting the apps I use, the tweaks I make, and how I stay focused in a bloated digital world. My interpretation of (almost) everything tech—quietly opinionated, always in progress.
