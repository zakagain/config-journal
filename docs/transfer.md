# 📤 File Transfer Tools

For local network transfers, I use  

[LocalSend](https://localsend.org)

— a simple, fast, and private way to send files between devices on the same network.

For remote transfers over longer distances,  

[Blip App](https://blip.net)

is my go-to—end-to-end encrypted and easy to use across platforms.

> Minimal setup, maximum reach.
